## CodeWork / React Native
- In this project, I created a simple job search application.
- I created the Jobs and Favorites pages using the **bottom tab navigation**.
- Using **Stack navigation**, I created the detail page so that when we click on a job on the main page, we can go to the detail page.
- I added icons to bottom tab navigation using React native **vector icons** package.
- I pulled the data from an API using the **Axios** package.
- I used **Render HTML** to show the job detail section written in HTML on the screen in the data I captured.
- There are two buttons on the detail page where we can add the job to favorites or submit.
- The same job cannot be added to favorites more than once.
